## 步驟 1 - 下載 VMware 16 Pro

<Alert icon='🔗' title='連結' color='blue' radius='md'>
  [下載 VMware Workstation 16
  Pro](https://www.vmware.com/products/workstation-pro/workstation-pro-evaluation.html)
</Alert>

根據你的系統下載相應的版本

![下載 Workstation 16 Pro](</static/images/blog/how-to-install-and-activate-vmware-16/step-1(1).png>)

## 步驟 2 - 安裝 VMware 16 Pro

1. 按 Next

![第 1 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(1).png>)

2. 接受條款
3. 按 Next

![第 2-3 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(2-3).png>)

4. 如需啟用增強型鍵盤支持，請勾選 Enhanced Keyboard Driver
5. 勾選 Add VMware Workstation console tools into system PATH
6. 按 Next

![第 4-6 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(4-6).png>)

請根據需求勾選

- Check for product updates on startup: 表示當 VMware Workstation Pro 啟動時，檢查應用程式和已安裝軟體組件的新版本。
- Join the VMware Customer Experience Improvement Program: 加入 VMware 客戶體驗改進計劃

7. 按 Next

![第 7 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(7).png>)

8. 按 Next

![第 8 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(8).png>)

9. 按 Install

![第 9 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(9).png>)

<Alert icon='💡' title='提示' color='yellow' radius='md'>
  安裝過程，耐心等候
</Alert>

![安裝中](/static/images/blog/how-to-install-and-activate-vmware-16/installing.png)

10. 按 Finish

![第 10 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(10).png>)

至此 VMware Workstation Pro 16 的安裝已完成

## 步驟 3 - 啟用 VMware 16 Pro (可選)

輸入 License Key，選擇下列其中一個即可

```
ZF3R0-FHED2-M80TY-8QYGC-NPKYF
YF390-0HF8P-M81RQ-2DXQE-M2UT6
ZF71R-DMX85-08DQY-8YMNC-PPHV8
```

1. 按 Continue

![第 1-2 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-3(1-2).png>)

3. 按 Finish

![第 3 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-3(3).png>)

## 步驟 4 - 設置繁體中文 (可選)

<Alert icon='🔗' title='連結' color='blue' radius='md'>
  [下載
  繁中語言包](/static/files/how-to-install-and-activate-vmware-16/zh_TW.zip)
</Alert>

1. 右鍵點擊 VMware Workstation Pro 16，按開啟檔案位置

![第 1 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(1).png>)

2. 到 messages

![第 2 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(2).png>)

3. 把剛下載的 `zh-TW.zip` 解壓縮，然後複製 / 移動到 `messages` 裡

![第 3 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(3).png>)

4. 右鍵點擊 VMware Workstation Pro 16，按內容

![第 4 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(4).png>)

5. 在 "目標" 的最後加上 `--locale zh_TW`
6. 按 確定

![第 5-6 步](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(5-6).png>)

再次開啟 VMware 後，界面成功切換為繁體中文

![中文介面](/static/images/blog/how-to-install-and-activate-vmware-16/chinese.png)

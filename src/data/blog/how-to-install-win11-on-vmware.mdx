## 安裝 VMware

[閱讀文章 - 如何安裝和啟動 VMware Workstation 16 Pro](/blog/how-to-install-and-activate-vmware-16)

## 步驟 1 - 下載 Windows 11 ISO

<Alert icon='🔗' title='連結' color='blue' radius='md'>
  [下載 Windows 11
  ISO](https://www.microsoft.com/zh-hk/software-download/windows11)
</Alert>

1. 選擇 Windows 11 (multi-edition ISO)
2. 按 下載
3. 選擇你的語言
4. 按 確認

![](</static/images/blog/how-to-install-win11-on-vmware/step-1(1-4).png>)

5. 按 64-bit 下載

![](</static/images/blog/how-to-install-win11-on-vmware/step-1(5).png>)

## 步驟 2 - 建立 Windows 11 虛擬機

1. 按下 建立新的虛擬機

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(1).png>)

2. 選擇 預設
3. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(2-3).png>)

4. 選擇 安裝程式光碟映像檔案(iso)
5. 按下 瀏覽 來選擇你剛下載的 Windows 11 ISO
6. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(4-6).png>)

7. 選擇 Microsoft Windows
8. 選擇 Windows 10 x64 或 Windows 10 and later x64 (如有)
9. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(7-9).png>)

10. 輸入 虛擬機名稱，此處我命名為 Windows 11
11. 按下 瀏覽 並選擇你要安裝虛擬機的位置，此處我選擇了 E 槽
12. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(10-12).png>)

13. 輸入 最大磁碟大小，建議為 60 GB 或以上
14. 選擇 將虛擬磁碟儲存為單個檔案
15. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(13-15).png>)

16. 完成

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(16).png>)

## 步驟 3 - 設定虛擬機

1. 按下 編輯虛擬機設定

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(1).png>)

2. 選擇 記憶體
3. 選擇 4 GB 或以上
4. 確定

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(2-4).png>)

5. 按下 開啟此虛擬機

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(5).png>)

## 步驟 4 - 安裝 Windows 11

1. 選擇要安裝的語言
2. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(1-2).png>)

3. 按下 立即安裝

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(3).png>)

1. 選擇 我沒有產品密鑰 或者 你也可以輸入密鑰

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(4).png>)

5. 選擇你想安裝的版本
6. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(5-6).png>)

<Alert icon='💡' title='提示' color='yellow' radius='md'>
  如無出現以下畫面請跳到 步驟 19。

按下 <Kbd>Shift</Kbd> + <Kbd>F10</Kbd> 開啟 CMD

</Alert>

![](/static/images/blog/how-to-install-win11-on-vmware/cant-install.png)

7. 輸入 `regedit`

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(7).png>)

8. 到 `電腦\HKEY_LOCAL_MACHINE\SYSTEM\Setup`
9. 右鍵 Setup
10. 新增 > 機碼，並命名為 `LabConfig`

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(8-10).png>)

4. 新增 > DWORD (32-位元) 值

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(11).png>)

5. 命名為 `BypassTPMCheck`
6. 輸入值為 1
7. 確定
8. 關閉 regedit (登錄編輯程式)

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(12-15).png>)

9. 關閉 Windows 安裝程式
10. 是

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(16-17).png>)

11. 立即安裝

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(18).png>)

12. 接受條款
13. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(19-20).png>)

14. 選擇 自訂: 只安裝 Windows (進階)

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(21).png>)

8. 選擇 磁碟機 0
9. 下一步

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(22-23).png>)

<Alert icon='💡' title='提示' color='yellow' radius='md'>
  安裝過程，耐心等候
</Alert>

![](/static/images/blog/how-to-install-win11-on-vmware/installing.png)

<Alert icon='✍️' title='提醒' color='yellow' radius='md'>
  由於初始化 Windows 11 較為簡單，因此此處略過步驟
</Alert>

## 步驟 5 - 安裝 VMware Tools (可選)

1. 按下 虛擬機
2. 按下 安裝 VMware Tools

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(1-2).png>)

1. 打開檔案總管
2. 選擇 DVD 光碟機
3. 按下 setup.exe

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(3-5).png>)

4. 是

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(6).png>)

5. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(7).png>)

1. 選擇 Typical
2. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(8-9).png>)

1. Install

![](</static/images/blog/how-to-install-win11-on-vmware/step-5(10).png>)

稍等片刻，你便能以全螢幕使用 Windows 11

![](/static/images/blog/how-to-install-win11-on-vmware/fullscreen.png)

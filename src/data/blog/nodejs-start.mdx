## 安装软件

1. 下载[nodejs](https://nodejs.org/en/)

2. 下载之后直接 next

## 配置环境变量

### 配置全局安装的模块路径和缓存路径

1. 在 nodejs 根目录创建 node_global 和 node_cache 文件夹

2. 管理员身份运行 cmd，配置路径

```
npm config set prefix <node_global文件夹>
npm config set cache <node_cache文件夹>
```

### 配置环境变量

1. 系统变量新建

变量名：`NODE_PATH`

变量值：`node_modules地址`

2. 在系统变量 path 添加

`路径+nodejs\`（npm 命令）

`路径+nodejs\node_global`（vue 命令）

3. 判断环境变量配置成功

全局安装最常用的 express 模块 进行测试： `npm install express -g`

### 检测安装（最新）

1. `node -v` 查看 node 的版本

2. `npm -v` 查看 npm 的版本(新版的 node 安装自带安装 npm)
